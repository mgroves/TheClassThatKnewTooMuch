"The Class That Knew Too Much: Refactoring Spaghetti Code"

Code and slides

Note that there are files like "BaseballControllerRefactored1.cs" that are not included in the csproj. This is because instead of typing out all the refactoring during the session, I simply exclude the old file (e.g. BaseballController.cs) and include the refactored file. I keep doing that as I go along: the ultimate file in the sequence is the finished product.

You'll may want something to view/edit the SQLite data file. I happen to like SQLite Administrator: http://sqliteadmin.orbmu2k.de/
